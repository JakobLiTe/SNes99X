# SNes99X
A unofficial fork of Snes9x - Portable Super Nintendo Entertainment System (TM) emulator that focuses on accuracy rather than compatibility <br> First emulator project :)
<br> Passes all http://blargg.8bitalley.com/parodius/snes-tests/ (site needs waybackmachine, as its not up right now. The files i used are in the releases tab of my repo) cputests and spctests, while something like snes9x fails more than a handful
<br> <br>
# Info (Why i did this)
I was mad. one of the most used SNES emulators (snes9x) is inaccurate. My ADHD brain couldnt be at peace with that. It's probably fine for play, i suppose, but i ran a accuracy test and it failed so hard. <br>
So i cloned it and started working on it, i just kept working and finessing till there were no issues left. <br>
You might be asking "But just use bsnes", fair point, but i kinda didnt know about it so i dont know. <br>
I will proably use bsnes, i dont have good documentation but this is more like a test project i did for fun on a wednesday night.
<br><br>
# USAGE
It's precompiled so either clone or download the zip from the <b>releases</b> tab.
Navigate to snes9x/libretro/ and import the core (snes99x_libretro.dylib) into retroarch, idk if it works on windows/linux, i mainly built the retroarch core for macOS. Feel free to try though.
